# 字幕术语表

此目录是 GUI 的默认术语表目录。程序会扫描本目录下除 `template_*.json` 之外的 JSON 文件，并在字幕页的“术语表”下拉框中显示。

规则：

- 一个 JSON 文件对应一种题材或一套可复用语境。
- 当前格式要求每个词条至少包含 `zh`、`en`、`ar`。
- `aliases` 用于补充 OCR、英文字幕或常见异写，不要放无关词。
- `note` 用于限定语境，防止同一个词在现代剧和历史剧中被错误套用。
- 只有在 GUI 中手动选择后，术语表才会注入初译和审核 Prompt；默认“不使用术语表”。
- 添加或修改文件后，在 GUI 中点击术语表旁的“刷新”。

可复制 `template_zh_en_ar.json` 创建新题材术语表。文件必须保存为 UTF-8 JSON。

内置术语表：

- `chinese_history_zh_en_ar.json`：中国历史、宫廷、科举及古装题材。
- `western_fantasy_zh_en_ar.json`：西方魔幻、魔兽契约、魔法学院、龙族及冒险题材；不收录普通角色名和地点名。
